== Description ==

Create a notification bar when customer has order over a specify amount. You can show a promotion like freeship or coupon code for customer.

== Installation ==

<h3>From your WordPress dashboard</h3>

<ol>
<li>Visit ‘Plugins > Add New’</li>
<li>Search for ‘c4d woo condition bar'</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
 

<h3>From WordPress.org</h3>

<ol>
<li>Download Woo Countdown Sale Product.</li>
<li>Upload the ‘c4d woo condition bar’ directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)</li>
<li>Activate plugin from your Plugins page.</li>
</ol>

<h3>Shortcode</h3>

[c4d-woo-cb amount="100"]

<h4>Other params</h4>

<p><strong>amount</strong>: order amount to get promotion. Default is 100.</p>
<p><strong>text</strong>: promotion text</p>
<p><strong>achive</strong>: achive text</p>
<p><strong>success</strong>: success text</p>
